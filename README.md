# Matplotlib Book

A practical guide to Matplotlib, from beginner concepts to advanced visualization techniques.

Topics include:

* Plotting fundamentals
* Object-Oriented API
* Subplots and layouts
* Colormaps and colorbars
* Scientific visualization
* Publication-quality figures

All figures in this book are generated using custom **`.mplstyle`** files included in this repository.

```python
plt.style.use("publication.mplstyle")
```

Written in LaTeX.
